#### Kursk State University Scientific Magazine App.

 - React, Redux Toolkit
 - React Bootstrap, Formik, i18next, Styled Components
 - ESLint
 - Webpack

____

#### install deps
```sh
npm install
```

#### run app
```sh
npm run start
```

#### build app in the development mode
```sh
npm run build
```

#### build app in the production mode
```sh
npm run build-prod
```

#### clean `dist` folder
```sh
npm run clean
```
